Pentaho Server CE e Client Tools

Pentaho Server CE

Provisioning on-demand di Pentaho BI Server CE (Community Edition) versione 5.1 con repository PostgreSQL 9.3. I dati del database vengono memorizzati su un volume della dimensione specificata dall’utente. L’utente potrà accedere alla user-console (Web UI per amministrare il server) tramite l’URL http://host:8080/pentaho usando l’account "admin" con la password specificata nel form di richiesta del servizio.

Versione:
5.1.0
