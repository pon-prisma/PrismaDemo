API per l'invio dei messaggi:

Metodo URL Descrizione
POST /biz/rest/SMSaas/sendSms/{workgroup} Invio sms:
  • workgroup: workgroup di appartenenza;
Body:
  • password: associato all'utente è visualizzabile nella schermata informazioni;
  • text: testo del messaggio;
  • num: numero di cellulare del destinatario;
  • scheduling: true se si vuole schedulare l'invio, false se si desidera inviare immediatamente;
  • sendDate: data in cui si desidera schedulare l'invio del messaggio(necessario settare a true il parametro scheduling), il formato da utilizzare è YYYY-MM-DD hh:mm:ss;
  • from: mittente del messaggio.
